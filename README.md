## Functionality


**metrics-switchvox-pbx**

Collects Metrics via SNMP for a Switchvox PBX
  
  - Disk
    - Used
    - Total
  - Load
  - Memory
    - Used
    - Total
  - Swap
    - Used
    - Total
  - Extentions
    - Used
    - Max
  - Calls
    - Total
    - Agent login
    - Agent logout
    - IVR
    - Parked
    - Queued
    - Ringing
    - Talking
    - Unknown
  - Sucscription Days Left
